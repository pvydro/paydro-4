Project Summary

Via wanted a interactive advertisement campaign, with a gamification twist, in the form of a PixiJS spin-the-wheel game, rewarding the player with a random promotional offer for the Via rideshare mobile app. Backend logic was then used to prevent the user from obtaining multiple offers, interacfing with a client database, via NodeJS running on a Google Cloud instance.

I was trusted with working on the entire application, frontend and backend, and have maintained & changed it since, under a retainer. My frontend work on this project showcases a combination of HTML5 web app design, and drawing with WebGL using PixiJS as a renderer.

Design

Design was done with Gimp and Photoshop. Animations were created using Gimp for hand-drawn spritesheets. Gimp is my go-to option for pixel art designs.

Development

Development was done with Java and LibGDX, with LibGDX offering cross-platform capabilities and a handful of libraries.

Platforms

Ammo Inferno was developed with mobile (iOS & Android) in mind. It also runs on PC, Mac & Linux, with LibGDX granting cross-platform capabilities.

Distribution

Distributed on the Google Play Store. For distribution, I used ASO techniques, showcasing store-oriented marketing skills.

Monetization

Interstitial ads, video ads, and banner ads were implemented via Google AdMob.
